 # Unwnd

Unwnd is a mobile app that helps users discover cool hangout spots in Nairobi 

## Features
- Browse curated places
- View ratings and descriptions
- Explore categories like Chill, Food, Nature

## Tech Stack
- Kotlin
- Jetpack Compose
